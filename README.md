# WitnessSnap

**WitnessSnap** is a privacy-first civic tech platform for anonymous reporting and family reunification. The platform allows witnesses to submit information about loved ones who were taken by immigration services or other urgent situations, while keeping user identities private by default. Optional accounts allow tracking of submissions over time.

## Features

- Anonymous submissions by default
- Optional account creation for tracking submissions
- Search functionality to locate loved ones
- Media uploads (photos/videos)
- Donation & mailing list integration (anonymous-friendly)
- Mobile-first design, responsive layout
